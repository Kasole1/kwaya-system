📋 1. INTRODUCTION

🎯 System Purpose

Kwaya Mt. Bonifasi System is a web-based application built with Python Flask for managing choir activities including members, fees, attendance, income, expenses, and media.

🔧 Technologies Used

  • Backend: Python 3.10+, Flask 2.3.3
  • Database: SQLite3
  • Frontend: HTML5, CSS3, JavaScript, Jinja2
  • Charts: Chart.js
  • Export: CSV, Excel (openpyxl)

🗄️ 5. DATABASE SCHEMA

Main Tables:

  • wanakwaya - Members
  • ada_calendar - Fees calendar
  • ada_monthly - Monthly fees
  • watumiaji - Users
  • mahudhurio_detailed - Attendance
  • mapato_sources - Income sources
  • expenses - Expenses
  • nyimbo - Songs
  • albamu - Albums

ada_calendar imelipwa values:

  • 0 = Hajalipa (Red)
  • 1 = Imelipwa kamili (Green)
  • 2 = Imelipwa sehemu (Yellow)
  • 3 = Ziada/Overpayment (Blue)

🔒 9. SECURITY GUIDELINES

  • Change default admin password immediately
  • Use strong passwords (8+ characters)
  • Regular backups (daily recommended)
  • Monitor activity log for suspicious activity
  • Limit user roles to minimum required permissions
